###### 安装vue-cli 3(2的升级版)

```bash
npm install -g @vue/cli
```

###### 创建项目

```bash
vue create my-project
```

###### 构建项目

```bash
npm run build
```

###### 运行项目

```bash
npm run serve
```

###### 如果想使用脚手架2版本(先拉取2.x模板)

```bash
npm install -g @vue/cli-init
```

###### vue-cli2 初始化项目

```bash
vue init webpack my-project
```

###### 查看版本

```bash
vue --version
```
